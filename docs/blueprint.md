# **App Name**: AgileSuit

## Core Features:

- Google Authentication: Allow users to log in using their Google account via Firebase Authentication.
- Email/Password Authentication: Enable users to sign up and log in with their email address and password, managed by Firebase Authentication.
- Forgot Password: Implement 'Forgot Password' functionality for users who have an account but have forgotten their password, handled through Firebase Authentication.
- Sign Up: Provide 'Sign Up' button to register on the website, handled through Firebase Authentication

## Style Guidelines:

- Primary color: Sky blue (#7CC9F1), reflecting the agile methodology with its association with open communication and clarity.
- Background color: Very light desaturated blue (#F0F8FF) to provide a calm and professional backdrop.
- Accent color: Light cyan (#B0E57C) for interactive elements to highlight essential features, creating contrast.
- Body and headline font: 'PT Sans', a sans-serif for a clear, accessible reading experience. 
- A clean and modern design in the style of high-end SaaS platforms.
